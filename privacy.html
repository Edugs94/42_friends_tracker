Privacy Policy – 42 Friends Tracker

Last updated: February 2026

42 Friends Tracker is an open-source project designed to respect your data and provide transparency.

1. Data Collection & Usage

This extension collects only the specific data required to function, accessed strictly via the official 42 School API.

  • Profile Data: We fetch your login, intra profile picture, and cursor level to display your user status.
  • Friends & Location: We fetch the location and exam status of the users you explicitly add to your "Friends" or "Exam" lists.

We do not track your browsing history, collect usage analytics, or sell any data to third parties.

2. Authentication & Security

Authentication is handled securely using the industry-standard OAuth2 protocol via 42 API.

  • The extension never sees, stores, or transmits your 42 password.
  • Authentication tokens are exchanged securely through a dedicated backend worker and stored locally in your browser.

3. Data Storage

All data (friend lists, cached profiles, and settings) is stored locally on your device using Chrome's storage.local and storage.sync APIs.

If you remove the extension, all locally stored data is immediately deleted from your browser.

4. Contact & Open Source

This project is open source. You can inspect the source code to verify these claims.

If you have any questions regarding this policy, please contact the developer via the support tab on the Chrome Web Store.
